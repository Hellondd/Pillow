# PyWatermarker

Модульный Python-инструмент для пакетной обработки изображений, наложения водяных знаков и управления метаданными на базе библиотеки Pillow (PIL).

## Используемые возможности Pillow

В проекте задействовано более 15 ключевых методов и классов библиотеки, распределенных по модулям:

1.  **Геометрия и Загрузка (`core/utils.py`)**:
    * `Image.open`: Неблокирующее чтение дескриптора файла.
    * `ImageOps.exif_transpose`: Автоматическая коррекция ориентации снимка на основе датчиков камеры.
    * `img.thumbnail`: Адаптивное изменение размера с сохранением пропорций и использованием фильтра `LANCZOS`.
    * `img.mode`: Доступ к цветовой модели.

2.  **Сложная Рисование (`core/watermark.py`)**:
    * `Image.new('RGBA')`: Генерирование полностью прозрачных холстов в оперативной памяти.
    * `ImageFont.truetype`: Загрузка и рендеринг векторных шрифтов (TTF).
    * `ImageDraw.Draw`: Инициализация контекста рисования.
    * `draw.textbbox`: Расчет геометрии текста до его нанесения (необходимо для центрирования).
    * `draw.text`: Рендеринг текста с поддержкой RGBA-цветов.
    * `Image.alpha_composite`: Математически корректное смешивание двух RGBA-изображений.

3.  **Композиция и Трансформация (`core/watermark.py`)**:
    * `img.paste(..., mask=...)`: Наложение графического логотипа с использованием его собственного альфа-канала в качестве маски прозрачности.
    * `logo.rotate(..., expand=True)`: Вращение логотипа на произвольный угол с расширением холста, чтобы углы не обрезались.

4.  **Сохранение и Анализ (`main.py`, `core/utils.py`)**:
    * `img.getexif`: Чтение бинарных метаданных камеры.
    * `ExifTags.TAGS`: Модуль декодирования числовых ID тегов в понятные строки (например, 'ApertureValue').
    * `img.convert('RGB')`: Конвертация цветовых пространств перед финальным сохранением.
    * `img.save()`: Экспорт в JPEG с применением оптимизации (`optimize=True`) и включением прогрессивной загрузки (`progressive=True`).
